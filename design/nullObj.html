Null Object

Forces of Conflict

When an edge case for a system needs to be easily detected without the excessive use of conditional statements, such as iterating along a path of objects until a null instance is reached.

Resolution

Create a subclass for each class that has the possibility of edge cases, where this subclass is a 'null' version of its superclass. The null class will serve as an interface, implementing the same methods and instance variables as the superclass, but using different implementations as to respond differently when accessed.

			public interface Example{
				bool isNull();
				string printData();
			}

			public class ExampleObject implements Example {
				private string data;

				public bool isNull() {
					return false;
				}
				public string printData() {
					System.out.println(data);
				}
			}

			public class NullObject implements Example {
				public static bool isNull() {
					return true;
				}
				public static string printData() {

				}
			}
